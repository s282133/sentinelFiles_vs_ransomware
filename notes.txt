hashCreator.py è uno script di prova per vedere se riesco a creare l'hash dei file.
Devo ampliarlo in maniera tale da lanciarlo solo una volta, immediatamente dopo la creazione delle sentinelle,
e deve agire su tutte le cartelle del file system.

L'idea migliore che ho avuto per ora è creare il file degli hash contenente gli hash di tutti i file in una data directory 
ma mantenere il numero delle sentinelle in cima al file. La creazione degli hash delle sentinelle dovrebbe avvenire prima 
rispetto a quella degli altri file.

ex "hashes.txt":
    3
    sentinella1.txt : hash1
    sentinella2.txt : hash2
    sentinella3.txt : hash3
    production1.txt : hash4
    production2.txt : hash5
    production3.txt : hash6
    production4.txt : hash7

in questo modo si mantengono gli hash di tutti i file, sentinella e non.
